Trust note 2

What support covers

Use support for access problems, broken delivery links, buyer clarification, or light workflow questions — not for custom implementation or private account changes.

Support request checklist

Support request checklist

  • Send the order ID or receipt reference first.
  • Name the product and the checkout or access platform used.
  • Describe the issue briefly and include any relevant screenshot, missing-file detail, or error text.

1

What to send

Send support requests to support@thewatchersedge.ca with enough context to reproduce or verify the issue.

  • Order ID or receipt reference
  • Product name and version if visible
  • Platform used for purchase or access
  • Short description of the issue and what you already tried

2

What support covers

Support is intended for access problems, broken delivery links, product clarification, and basic workflow questions. It does not include custom implementation, private account changes, or professional legal, tax, or accounting advice.
